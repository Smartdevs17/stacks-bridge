# Stacks Bridge - Bitcoin Peg-in/Peg-out

Bitcoin bridge contracts for Stacks blockchain using @stacks/transactions.

## Contracts

- `btc-peg-in-v1.clar` - Bitcoin peg-in functionality
- `btc-peg-out-v1.clar` - Bitcoin peg-out functionality

## Deployment

Deployed to mainnet at `SP9AS5B36MKC0FVF4DE75A1EBPANXQ14AEH98BH0`

## Usage

```typescript
import { makeContractCall } from '@stacks/transactions';
```
